# 更多输入→输出示例

## 电商类

**输入：** 做一个小红书风格的种草电商 App

**输出：**
```markdown
创建完整的小红书风格种草电商App。
全局风格：卡片流、大图、圆角、浅色主调、标签与话题
需要以下页面/模块：
【首页】信息流、双列瀑布流、点赞/收藏/评论、话题标签
【笔记详情】大图、正文、商品链接、作者信息、互动
【发布】多图、文字、话题、@、位置
【搜索】搜索框、热搜、历史、分类（笔记/用户/商品）
【消息】点赞/评论/关注、私信
【我的】头像、关注/粉丝、收藏/赞过、订单、设置
【商品/店铺】商品列表、加购、下单入口
```

## 社交类

**输入：** 做一个类似 Discord 的社群/频道聊天应用

**输出：**
```markdown
创建完整的 Discord 风格社群聊天App。
全局风格：深色模式、侧边栏、圆角、频道列表、角色/权限视觉
需要以下页面/模块：
【服务器/社群】侧边栏频道列表、文字/语音频道、成员列表
【频道聊天】消息流、输入框、表情、引用、@、附件
【语音频道】加入/离开、静音、扬声器、成员状态
【私信】会话列表、单聊消息、已读
【发现】探索服务器、推荐、搜索
【个人/设置】头像、状态、通知、隐私、账户
```

## 工具类

**输入：** 做一个 Notion 风格的笔记与知识库 App

**输出：**
```markdown
创建完整的 Notion 风格笔记与知识库App。
全局风格：侧边栏、块编辑、拖拽、浅色/深色切换、简洁图标
需要以下页面/模块：
【侧边栏】工作区、页面树、快捷方式、回收站
【页面编辑】标题、块（段落/标题/列表/代码/表格/嵌入）、拖拽排序、斜杠命令
【数据库/表格】多视图（表格/看板/日历）、属性、筛选排序
【搜索】全文搜索、最近、收藏
【设置】账户、主题、协作、导入导出
```
